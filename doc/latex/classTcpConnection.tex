\doxysubsection{Detailed Description}
\mbox{\hyperlink{classTcpConnection}{Tcp\+Connection}} 是服务器与客户端之间连接的抽象。 它负责管理 socket fd，并通过 \mbox{\hyperlink{classChannel}{Channel}} 将其注册到 Event\+Loop。 它拥有 input\+Buffer 和 output\+Buffer，用于处理读写事件。 它是 noncopyable 的，因为每个 \mbox{\hyperlink{classTcpConnection}{Tcp\+Connection}} 对象都代表一个唯一的连接。
