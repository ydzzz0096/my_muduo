\doxysubsection{Detailed Description}
\mbox{\hyperlink{classAcceptor}{Acceptor}} 类的职责是封装服务器端的监听套接字。 它在构造时创建 socket，并设置 socket 选项。 它不拥有 Event\+Loop，而是通过指针使用。
