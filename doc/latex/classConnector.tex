\doxysubsection{Detailed Description}
负责主动发起 TCP 连接的类。 核心逻辑： 


\begin{DoxyEnumerate}
\item 调用 \+::connect。
\item 如果返回 EINPROGRESS，则注册 \mbox{\hyperlink{classChannel}{Channel}} 监听 EPOLLOUT (可写) 事件。
\item 当 \mbox{\hyperlink{classChannel}{Channel}} 可写时，通过 getsockopt(\+SO\+\_\+\+ERROR) 检查连接是否真的成功。
\item 如果成功，回调 New\+Connection\+Callback；如果失败，执行重连逻辑。 
\end{DoxyEnumerate}
