Detailed Description

TcpConnection 是服务器与客户端之间连接的抽象。 它负责管理 socket fd，并通过 Channel 将其注册到 EventLoop。 它拥有 inputBuffer 和 outputBuffer，用于处理读写事件。 它是 noncopyable 的，因为每个 TcpConnection 对象都代表一个唯一的连接。
