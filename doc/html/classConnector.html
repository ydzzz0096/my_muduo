Detailed Description

负责主动发起 TCP 连接的类。 核心逻辑：

  1. 调用 ::connect。
  2. 如果返回 EINPROGRESS，则注册 Channel 监听 EPOLLOUT (可写) 事件。
  3. 当 Channel 可写时，通过 getsockopt(SO_ERROR) 检查连接是否真的成功。
  4. 如果成功，回调 NewConnectionCallback；如果失败，执行重连逻辑。
